# Calculadora POO — [Denise Juarez]

## Descripción
Calculadora desarrollada en Python con Tkinter aplicando 
el paradigma de Programación Orientada a Objetos (POO).

## Tema visual elegido
Elegi el Tema 4, en mi caso elegi algo pastel y retro, siento que es algo representa las cosas que me gustan.

## Estructura del proyecto
- `calculadora.py` — Archivo principal
- `README.md`      — Este archivo

## Clases principales
- `Calculadora` — Contiene la lógica de cálculo
- `Interfaz`    — Gestiona la interfaz gráfica con Tkinter

## Cómo ejecutarlo
```
python calculadora.py
```

## Tecnologías
- Python 3.x
- Tkinter (biblioteca estándar)
